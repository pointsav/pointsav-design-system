# dtcg-vault — plantilla del vault

[ 🇬🇧 Read this document in English ](./README.md)

Este directorio es la **plantilla canónica** para un vault por
inquilino bajo la afirmación doctrinal #38 (Substrate del Sistema
de Diseño). Es el contenido que `app-privategit-design` lee desde
disco en tiempo de ejecución — distinguido del árbol previo en
formato YAML que reside en los directorios raíz `tokens/`,
`themes/`, `components/`, `templates/`, `guidelines/`, `docs/` y
que es consumido por clústeres hermanos (`project-orgcharts`).

Documentación operativa completa en `README.md` (inglés).
